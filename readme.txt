Simple backups and a powerful Recovery Portal to fix your site when it crashes.

== Description ==

SafeBackup & Repair WP provides peace of mind with simple local backups and a unique generic "Recovery Portal" that allows you to access your site even when wp-admin is broken (WSOD).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/safebackup-repair-wp` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the SafeBackup menu to configure settings and creating backups.
